# OopsiePrevent
Backup a folder locally in case of accidentally deleting contents

Created this after accidentally deleting tons of photo mode content for a game after re-installing it.
Backing up locally is easy, but I wanted something that removed some of the tedium/friction which caused me to not do it in the first place.

This tool let's you simply select the source and destination folders and click one button to copy everything over.
The app will save history of folders selected, and will only copy new files on subsequent uses.

Can be run from command line but to run as a more simple Windows App just create a shortcut to the file, then edit the properties and prepend pythonw.exe before the path in the Target field.
